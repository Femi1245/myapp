Music in the game is streamed via the YouTube IFrame API. Each level uses a
`youtubeId` in `app/page.tsx` (SONGS array and `DASHBOARD_YT_ID` for the menu).

The files listed below are not required at runtime unless you switch the code
back to local HTMLAudio:

- dashboard.mp3
- faded.mp3
- unity.mp3
- stay.mp3
- closer.mp3
- perfect.mp3
- all_of_me.mp3
- blinding_lights.mp3
- counting_stars.mp3
- shape_of_you.mp3
- see_you_again.mp3
